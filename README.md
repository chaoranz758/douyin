# 极简版抖音
## 一句话介绍
实现了基于微服务的极简版抖音的基础接口、互动接口、点赞接口，使用mysql存储数据，基于redis、rocketmq等组件对系统做了性能优化，增加了部分安全功能，并对系统做了功能测试和性能测试。
